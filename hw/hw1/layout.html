Layout and re usage

Understand why component based layout is good to use

As we have learned from previous sections, web components can be created and isolated. Placed where it fits and can togheter create complex and re usable interfaces. It is important to understand the importance of how this method greatly increases the productivity of developers, both in development but also in the maintaining phase. Bellow is a list of five keywords and sentences you should keep in mind when developing a web component based user interface.

  • Organize
  • Build And Test
  • Demonstrate And Improve
  • Write Good Documentation
  • Release

The image above is an example of a component based application. As we can see from the image, the application is splitted into seperate parts. Each part is representing a re usable component. These parts are together forming the working application. Since the components are re usable, we can easily insert a searchbox wherever we want, or maybe even the entire form if we like.
