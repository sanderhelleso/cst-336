Popular frameworks

Get an insight into some of the most popular frameworks

Today, there are alot of popular frameworks out there that builds on the the idèa of web components. Frameworks allow developers to easier structure code and set state and functionality to each seperate component. We can import and export as we like. Some of the frameworks i will be suggesting to check out are: React.js, Vue.js and Angular. These are the current component based frameworks that are in most demand. They are all based on components and use JavaScript as its langauge.

Above we can see the three menitioned frameworks. These frameworks makes it painless to create interactive user interfaces. They allow developers to design simple views for each state in the application, the framework will efficiently update and render just the right components when your data changes. The following quote is taken from the React.js webpage and perfectly explains a huge benefit of using react or one of the other mentioned:



Build encapsulated components that manage their own state, then compose them to make complex UIs. Since component logic is written in JavaScript instead of templates, you can easily pass rich data through your app and keep state out of the DOM.
